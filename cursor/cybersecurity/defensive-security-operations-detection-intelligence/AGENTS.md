# Cursor Area 05 Instructions

Work only inside `cursor/cybersecurity/defensive-security-operations-detection-intelligence/`.

Use supplied static evidence to create SOC operating models, telemetry requirements, logging coverage maps, detection specs, validation plans, lifecycle records, alert triage records, hunt plans/reports, intelligence assessments, defensive malware-analysis plans, SOAR playbook designs, coverage assessments, KPI/KRI definitions, quality reviews, escalation records, and assurance records.

Declared incidents transfer primary ownership to Incident Response, DFIR, and Recovery.
