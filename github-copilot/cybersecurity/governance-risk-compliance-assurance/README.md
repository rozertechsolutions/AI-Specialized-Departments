# GitHub Copilot Cybersecurity Governance, Risk, Compliance & Assurance

This package defines a GitHub Copilot-native Cybersecurity GRC & Assurance specialization.

## Components

- `.github/copilot-instructions.md`: repository custom instructions for this area.
- `.github/agents/*.agent.md`: repository custom agents for specialist ownership.
- `.github/skills/*/SKILL.md`: project agent skills for reusable GRC workflows.
- `reference/NATIVE_SOURCES.md`: official GitHub documentation basis and omitted-component decisions.

Hooks, MCP servers, extensions, scripts, active automations, and live integrations are intentionally omitted.
