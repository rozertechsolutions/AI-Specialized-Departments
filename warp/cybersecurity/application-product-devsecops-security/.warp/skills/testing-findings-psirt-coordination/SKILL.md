# Testing Findings PSIRT Coordination

## When To Use

Use for application security testing strategy, finding triage, validation planning, PSIRT intake, and disclosure decision packages.

## Instructions

1. Confirm testing scope, allowed methods, severity model, affected assets, owners, and disclosure boundaries.
2. Plan evidence requirements, triage findings, define validation criteria, and draft PSIRT package content.
3. Record human approvals required for disclosure and closure.
4. Do not test actively, contact external parties, disclose, command incidents, or close findings.
