# Junie Area 05 Guidelines

Use static evidence only for SOC operating models, telemetry requirements, logging coverage maps, detection specs, validation plans, lifecycle records, alert triage, hunts, intelligence, defensive malware-analysis planning, SOAR design, coverage, metrics, quality review, escalation, and independent assurance.

Do not connect to security systems, run live queries, execute hunts, deploy rules, close alerts, isolate hosts, block indicators, delete data, contain incidents, access malware, execute malware, authenticate, install dependencies, run generated files, publish, or push.
