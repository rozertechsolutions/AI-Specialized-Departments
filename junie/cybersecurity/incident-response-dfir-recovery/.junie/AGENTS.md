# Junie Area 06 Guidelines

Use static evidence only for incident readiness, classification, command support, evidence governance, forensic planning, containment and eradication options, secure recovery, scenario coordination, crisis handoff, tabletop exercises, post-incident reviews, corrective actions, and independent assurance.

Do not execute files, authenticate, connect integrations, access systems, collect evidence, run forensic tools, contain threats, restore services, rotate credentials or keys, notify externally, negotiate ransom, declare incidents, accept residual risk, or close incidents.
