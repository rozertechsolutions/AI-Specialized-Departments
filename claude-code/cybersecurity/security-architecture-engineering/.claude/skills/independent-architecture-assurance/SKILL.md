# independent-architecture-assurance

1. Confirm reviewer independence and source set.
2. Trace each material claim to evidence.
3. Check threat coverage, control sufficiency, resilience, ownership, assumptions, residual risk, approval path, and implementation ambiguity.
4. Classify findings as blocker, required correction, advisory improvement, or residual limitation.
5. Return readiness statement without accepting risk, approving architecture, or closing findings.
