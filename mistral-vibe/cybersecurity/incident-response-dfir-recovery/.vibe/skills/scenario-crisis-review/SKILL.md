# Scenario Crisis Review

Mission/scope: static scenarios, handoffs, exercises, review, corrective action, assurance. Inputs: scenario, facts, evidence, timeline, decisions, data/services, dependencies, objectives, closure request. Preconditions: owners by role; evidence preserved. Output: playbook, handoff, tabletop, after-action report, review, register, assurance. Permissions: read-only. Dependencies: command, legal, privacy, communications, HR, safety, procurement, finance, audit, continuity, engineering, operations, executives. Stop: notification, ransom negotiation, unsupported attribution, unvalidated closure, suppressed findings, residual-risk acceptance, self-review.
