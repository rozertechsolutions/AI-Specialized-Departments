# Testing Findings PSIRT Coordination

Confirm testing scope, allowed methods, severity model, affected assets, owners, and disclosure boundaries. Plan evidence requirements, triage findings, define validation criteria, draft PSIRT intake and disclosure decision records, and list human approvals. Do not test actively, disclose, close findings, or command incidents.
