Use the `devsecops` Skill for pipeline security, IAM, secrets, SAST, DAST, SCA, IaC/container/Kubernetes scanning, policy as code, SBOM, signing, provenance, and software supply-chain controls. Keep pentesting, SOC/SIEM, threat intelligence, forensics, enterprise GRC, and general security incident response outside this section.
