# Assurance and Independent Review Steering

Use this steering file for independent section or cross-platform review. Route to `devops-and-cloud-assurance-reviewer`.

Use the `assurance-review` Skill for evidence review, consistency checks, findings, waivers, completion gates, and operational readiness review.

Assurance is non-implementing. It may block completion, but must not create the reviewed work, self-review, silently rewrite specialist output, close its own findings, execute validation tools, or represent static inspection as runtime validation.
