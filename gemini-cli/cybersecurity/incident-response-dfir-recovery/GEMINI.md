# Gemini CLI Area 06 Instructions

Work only inside `gemini-cli/cybersecurity/incident-response-dfir-recovery/`.

Mission: create static incident response, DFIR planning, and recovery assurance artifacts covering readiness, classification, command support, evidence governance, forensic planning, containment/eradication options, secure recovery, scenario coordination, crisis handoff, tabletop exercises, post-incident review, corrective actions, and independent assurance.

Do not execute generated files, authenticate, connect tools, access systems, collect evidence, run forensic tools, contain threats, restore services, rotate credentials or keys, notify externally, negotiate ransom, declare incidents, accept residual risk, or close incidents. Preserve provenance and require human authority for all final decisions.
