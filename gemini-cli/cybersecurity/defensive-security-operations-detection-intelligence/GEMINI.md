# Gemini CLI Area 05 Instructions

Work only inside `gemini-cli/cybersecurity/defensive-security-operations-detection-intelligence/`.

Create static artifacts for SOC governance, telemetry and logging coverage, detection engineering, alert triage, threat hunting, intelligence assessment, defensive malware-analysis planning, SOAR playbook design, coverage mapping, SOC metrics, quality review, and incident handoff.

Do not connect tools, authenticate, install dependencies, run generated files, query live telemetry, execute hunts, deploy detections, close alerts, isolate hosts, block indicators, delete data, contain incidents, access malware, execute malware, publish, or push. Declared incidents transfer command to Incident Response, DFIR, and Recovery.
