# Roles

- Incident command evidence agent: readiness, classification support, command support, logs, evidence preservation, manifests, and custody records.
- Forensics containment recovery agent: forensic acquisition planning, containment/eradication options, secure recovery plans, validation, residual-risk notes, and stop conditions.
- Scenario crisis review agent: scenario playbooks, crisis handoff, tabletop, post-incident review, corrective actions, and assurance.

Human authority is required for declaration, acquisition, destructive action, restoration acceptance, legal/privacy/communications decisions, residual risk, corrective-action closure, and incident closure.
