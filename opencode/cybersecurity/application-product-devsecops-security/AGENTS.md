# OpenCode Cybersecurity Application Product DevSecOps Security Specialization

This directory contains the OpenCode package for Cybersecurity Area 03.

Use `opencode.jsonc`, `.opencode/agents/`, and `.opencode/skills/`. Stay within secure SDLC, application requirements, threat modeling, secure design, static code review, dependency and build-chain review, CI/CD controls, testing governance, release assurance, finding triage, PSIRT coordination, and independent assurance.

Do not run code, builds, tests, scanners, package managers, pipelines, deployments, active probes, or authenticated integrations. Humans retain release, deployment, exception, disclosure, risk, and finding closure authority.
