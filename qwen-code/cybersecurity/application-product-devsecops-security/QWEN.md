This directory contains the Qwen Code package for Cybersecurity Area 03.

Use `.qwen/settings.json`, `.qwen/agents/`, and `.qwen/skills/`. Scope is secure SDLC, security requirements, threat modeling, abuse cases, secure design, static code review, dependency and build-chain review, CI/CD controls, testing governance, release assurance, finding triage, PSIRT coordination, and independent assurance.
