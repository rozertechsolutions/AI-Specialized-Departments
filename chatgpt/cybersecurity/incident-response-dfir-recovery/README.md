# ChatGPT Incident Response DFIR Recovery

Static ChatGPT Project and Custom GPT package for Cybersecurity Area 06. It provides project instructions, knowledge artifacts, reusable Skills, workflow guidance, output templates, and source references for incident response, DFIR planning, and recovery assurance.

No artifact in this package is executable. Do not connect apps, query live systems, collect evidence, contain threats, restore services, send communications, or make human-only decisions.
