# Detection Triage Hunting

Mission: design detection hypotheses and specifications, triage supplied alerts, and plan threat hunts without live queries or deployment.

Inputs: threat hypothesis, supplied alert or case data, required telemetry, asset context, rule version, timestamps, evidence, expected behavior, tuning context, hunt basis.

Deliverable: detection specification, validation plan, lifecycle record, alert triage record, hunt plan, confidence, rationale, escalation criteria, incident handoff decision, and completion criteria.

Stop for rule deployment, live queries, containment, automatic high-impact closure, running hunts, or continuing after incident declaration without Incident Response coordination.
