# Mobile Specialization Scope

- Native classification: `AGENTS.md`, `kilo.jsonc`, `.kilocodeignore`, `.kilo/agents/*.md`, `.kilo/skills/*/SKILL.md`, `.kilo/rules/*.md`, and `.kilo/plugins/*.js` are native Kilo Code project components.
- Unsupported and omitted: legacy `hooks/`, `mcp/`, `workflows/`, top-level `agents/`, top-level `subagents/`, active MCP startup, authenticated connectors, model pinning, publishing automation, signing automation, deployment automation, and universal adapters.
- Work only in this specialization directory.
- Discover actual project commands from the target mobile repository before running validation.
- Treat Android, iOS, KMP, Flutter, and React Native as separate ownership areas unless an existing repository already combines them.
