# Application Product DevSecOps Scope

Cover secure SDLC governance, application security requirements, threat modeling, abuse cases, secure design, static code review guidance, dependency and build-chain review, CI/CD controls, testing governance, release readiness, finding triage, PSIRT coordination, and independent assurance.

Do not own enterprise policy approval, general infrastructure architecture, SOC monitoring, incident command, broad vulnerability operations, production deployment, or unrestricted offensive testing.
