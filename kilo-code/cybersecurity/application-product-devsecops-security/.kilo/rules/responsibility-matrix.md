# Responsibility Matrix

- Governance agent: SDLC controls, ownership, gates, metrics, and training plans.
- Requirements and threat agent: requirements, misuse cases, trust boundaries, and abuse paths.
- Design and code agent: secure design review and static code-review guidance.
- Supply-chain and release agent: dependency, SBOM, provenance, build, CI/CD, and release controls.
- Testing and PSIRT agent: testing strategy, finding triage, remediation evidence, and product vulnerability coordination.
- Independent reviewer: impartial assurance and approval-readiness review.
