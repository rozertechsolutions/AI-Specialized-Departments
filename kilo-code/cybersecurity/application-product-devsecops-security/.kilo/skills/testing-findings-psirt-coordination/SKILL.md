# Testing Findings PSIRT Coordination

1. Confirm testing scope, allowed methods, artifacts, severity model, affected assets, owners, and disclosure boundaries.
2. Build a governance plan for static, dynamic, dependency, and manual review evidence without executing tests.
3. Triage findings by impact, exploitability indicators, affected versions, confidence, remediation owner, and due date.
4. Prepare PSIRT intake, reporter communication drafts, disclosure decision records, and validation criteria for human review.
5. Do not contact external parties, disclose issues, close findings, or perform incident command.
