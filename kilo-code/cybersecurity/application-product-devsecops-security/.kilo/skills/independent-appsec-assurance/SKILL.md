# Independent AppSec Assurance

1. Confirm independence, scope, source set, author, owner, reviewer, approver, and requested decision.
2. Check evidence quality, requirement coverage, threat model completeness, design and code-review rationale, dependency and release controls, testing governance, finding triage, and PSIRT readiness.
3. List approval blockers, residual risk, unsupported claims, missing evidence, and required human decisions.
4. Return approve-ready, conditionally ready, or not ready as advisory status only.
5. Do not certify, accept risk, approve release, or approve disclosure.
